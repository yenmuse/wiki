1dB 压缩点

首先，从理论上定义 1dB 压缩点 [1]。

在设计小信号放大器的时候，我们会认为放大器的增益恒定，放大器的输出随输入线性变化。但当输入功率增加到一定程度时，由于系统存在 非线性，放大器的输出不再随着输入线性变化。事实上，输出功率增加的速度会逐渐慢于线性增加的速度，输入 - 输出曲线会出现下图所示的弯折。当实际的输出功率比输入功率低 1dB 时，对应的曲线图上的点就称为 1dB 压缩点，对应的输入功率称为 输入 1dB 压缩点，对应的输出功率称为 输出 1dB 压缩点。

在收发系统中，通常就用 1dB 压缩点 来衡量系统的非线性。一般发射端的功率放大器会关注 输出 1dB 压缩点（最大能输出多少线性功率），而接收端的低噪声放大器会关注 输入 1dB 压缩点（最大能接收多少线性功率）。

Cadence 实测

可以采用仿真软件 Cadence 来测量 1dB 压缩点。因为担心版权问题，所以下面仅用文字描述需要注意的设置，如果想要阅读详细的步骤，可以参考 Cadence 官方的教程 [2]。

以二端口电路为例，打开原理图里的输入 port，将输入功率幅度 Amplitude 1 (dBm) 设置为变量 pin 。

打开 ADE L，在 Analyses - Choose 下选择 hb 仿真。

设置基频 Fundamental Frequency 为测量的中心频率，谐波个数 Number of harmonics 取值应大于 3，过采样因子 Oversample factor 笔者也不清楚用途，一般无脑地设为 8。

勾选 Sweep，选择变量仿真（不是频率仿真，变量个数为 1），设置变量为 pin。设置变量扫描范围 Sweep Range（Start-Stop 模式），Range 的起始点需要设低一些，因为仿真器默认起始点为线性，如果起始点设得比较高，此时电路的非线性已经比较明显，会使得仿真结果不准确。扫描种类 Sweep Type 可设为 Linear+Step Size，意思是线性取点，设置取点间隔（设为 1，2，3 等都可以）。设置完毕后开始仿真，等待片刻。

仿真完成后，在 Results-Direct Plot-Main Form 中勾选 Compression point。如果想知道输入 1dB 压缩点，就选择 Input Referred 1dB Compression Point；如果想知道输出 1dB 压缩点，就选择 Output Referred 1dB Compression Point。一阶谐波 1st Order Harmonic 设为中心频率。

Select 下拉框选中 Port(fixed R(port)) ，点击原理图中的输出 port，仿真器就会自动绘制输入 - 输出曲线图，并在图中标出 1dB 压缩点的值。
